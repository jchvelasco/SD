Basic Features

  • Multiple Cursors
  • Vintage Mode
  • Command Palette CTRL + SHIFT + P
  • Jump to matching bracket CTRL + M
  • Select word CTRL + D - (repeat to select the next occurence of the selected word)

Plugins

Sublime Package Control
A full-featured package manager that helps discovering, installing, updating and removing packages for Sublime Text 2. It features an automatic upgrader and supports GitHub, BitBucket and a full channel/repository system.
Tag
"Tag" plugin is a collection of packages about tags, mixed together in an effort to provide a single package with utilities to work with tags. Default Shortcut - CTRL + ALT + F
BracketHighlighter
Bracket Highlighter matches a variety of brackets such as: [], (), {}, "", '',, and even custom brackets.
SublimeLinter
SublimeLinter is a plugin that supports "lint" programs (known as "linters"). SublimeLinter highlights lines of code the linter deems to contain (potential) errors. It also supports highlighting special annotations (for example: TODO) so that they can be quickly located. Default Shortcut - CTRL + ALT + L
